# Student Profile Management System

## Overview

The Student Profile Management System is a Python application that allows users to manage student information efficiently. The system provides functionality to add, view, search, update, and delete student profiles through a simple command-line interface.

## Features

* Add new student profiles
* View all student profiles
* Search for a student by Student ID
* Update student information
* Delete student profiles
* Object-Oriented Programming (OOP) implementation
* Simple and user-friendly menu system

## Technologies Used

* Python 3
* Object-Oriented Programming (OOP)

## How to Run

1. Clone this repository:

```bash
git clone https://github.com/YOUR_USERNAME/student-profile-management-system.git
```

2. Navigate to the project directory:

```bash
cd student-profile-management-system
```

3. Run the Python file:

```bash
python student_profile_system.py
```

4. Follow the on-screen menu options to manage student profiles.
